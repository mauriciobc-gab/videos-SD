# videos-SD
 Workspace de motion graphics corporativos em HyperFrames para Soluções Digitais (DTI / Grupo Águia Branca). Produção multi-projeto de vídeos com resultados de RPA (2025/2026) e Análise Crítica Motorista/Implantação. Construído com HyperFrames + GSAP, 1920×1080, fontes Satoshi/Cabin auto-hospedadas.
